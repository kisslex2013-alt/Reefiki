# Public Backlog

## Русский

Это публичный backlog REEFIKI. Он не заменяет закрытый `TASKS.md`: private
backlog остаётся рабочим журналом разработки, а эта страница показывает
только внешне понятные направления.

| Priority | Направление | Что считается готовым |
|---|---|---|
| P0 | Чистый public snapshot | Public GitHub содержит install/use/product docs, tests, skills, adapters and demo; закрытые рабочие отчёты и local state исключены. |
| P0 | First-run clarity | Новый пользователь проходит install -> onboarding -> demo -> connect без знания внутренних правил. |
| P1 | Public docs consistency | README, COMMANDS, install, demo and roadmap docs не ссылаются на закрытые рабочие artifacts. |
| P1 | Agent portability | Base skills and adapters remain reusable across Codex, Claude Code, Cursor/Windsurf, Gemini, Mimo-style surfaces without auto-mutating user configs. |
| P2 | Evidence UX | Intent/source/runtime checks get clearer examples and small public demos. |

## English

This is the public REEFIKI backlog. It does not replace the private `TASKS.md`:
the private backlog remains the development ledger, while this page
shows only externally understandable work streams.

| Priority | Direction | Done Means |
|---|---|---|
| P0 | Clean public snapshot | Public GitHub contains install/use/product docs, tests, skills, adapters, and demo; private working reports and local state are excluded. |
