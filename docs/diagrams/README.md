# REEFIKI Diagrams

This folder keeps editable Draw.io source diagrams for REEFIKI architecture documentation.

## Files

| File | Purpose |
| --- | --- |
| `reefiki-l1-high-level-architecture.drawio` | L1 overview of users, agents, dashboards, CLI, core modules and durable stores. |
| `reefiki-l2-core-services-and-adapters.drawio` | L2 map of CLI services, `reefiki_core/*`, adapters, memory providers and publish/guard surfaces. |
| `reefiki-l3-project-workspace-dataflow.drawio` | L3 project workspace dataflow for `projects/<name>/`: capture, process, query, harvest, status and publish boundaries. |

## Usage Notes

- Treat `.drawio` files as editable source artifacts.
- Export stable `.svg` or `.png` assets before embedding diagrams in the public README.
- Do not add private project content, personal absolute paths, secrets or local machine names to these diagrams.
