# REAL TIME #006 - Канал о памяти не должен писаться по памяти

Не AGI подключён к REEFIKI через local bridge. Это значит, что у редакционного проекта есть слой памяти: можно восстановить, откуда взялась история, какие источники уже разобраны и какие решения нельзя потерять между сессиями.

REEFIKI здесь не автор и не автопубликатор. Она помогает держать source packs, решения и контекст так, чтобы следующий агент или следующая сессия не начинали с пустого места и не пересказывали тему "по ощущениям".
